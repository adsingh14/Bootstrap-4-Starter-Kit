## README

---
#### What's inside ?
* Offline & CDN version.
* Bootstrap 4.1.3
* Font Awesome 5.1 (include v4 support)
* jQuery
* Popper.js

#### How to install ?

Just clone the git via Terminal or Git Bash:

```git clone https://github.com/adsingh14/Bootstrap-4-Starter-Kit```

**OR**

Download the `.zip` file directly.
